# Check Today's Wordle (and future Wordles!)

## Website

📝 Today's  Wordle - https://djsdev.github.io/tomorrows-wordle/

## 🏦 Start Locally

```sh
npm run build
```

## .env

Create `.env` in root directory

```
WORDLE_PORT=8080
WORDLE_HOSTNAME=0.0.0.0
WORDLE_LOG_BODY=false
WORDLE_DEV_ENV=false
```
